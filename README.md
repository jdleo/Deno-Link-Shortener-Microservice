## Deno Link Shortener Microservice

This app is a POC of a microservice for a link shortener.

**Technology**

-   API: Deno, Typescript
-   Infrastructure: Docker, Docker Swarm
-   Database: PostgreSQL

**Build & Run Swarm**

```
docker-compose -f stack.yml up --build
```

**Run Swarm**

```
docker-compose -f stack.yml up
```

**API Base URL**

```
localhost:3001/
```
